遊戲流程

遊戲分為五層，每一層有15個房間
必須打贏至少4個關卡才能進入Boss房間
必須打贏每一層 Boss 後才能進到下一層
最終打贏第五層的 Boss 後能上傳紀錄

遊戲玩法

每一個房間都需要回答一個資訊安全相關問題
答錯了會影響玩家屬性，答錯越多影響越大
遊戲中有不同方面的升級手段 : 成長系統
透過打倒怪物所得到的金幣來強化玩家各項數值
遊戲的玩法為 Rougelike
每一局新遊戲玩家的各項技能等級皆為從零開始
玩家能自由選擇自己最喜歡的技能組合來遊玩

遊戲技能設定

技能分成三個階級
階級一消耗能量為5，升級所需技能點為5
階級二消耗能量為25，升級所需技能點為25
階級一消耗能量為125，升級所需技能點為50
每個技能最高都可以升到7級

階級一技能

劍氣


射出一道劍氣，以31.25格/秒飛行1秒
傷害為基礎攻擊的3 / 3.1 / 3.2 / 3.3 / 3.4 / 3.5 / 3.6倍
擊退力度為4，冷卻時間1.5秒
